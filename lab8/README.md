# Лабораторная работа 8. Ансамблевые модели и методы классификации
Данная лабораторная работа посвящена изучению и практическому применению ансамблевых методов машинного обучения (случайный лес, бэггинг, градиентный бустинг) и сравнению различных алгоритмов классификации и регрессии. В работе рассматриваются задачи настройки гиперпараметров, использования конвейеров (Pipeline), обработки пропущенных данных и работы с популярными библиотеками градиентного бустинга.

## Краткая инструкция по запуску
Проект состоит из набора Jupyter-блокнотов, расположенных в директории `src/`. Для их корректной работы необходимо настроить виртуальное окружение.

### 1. Настройка окружения
1.  Установите Anaconda Distribution или Miniconda.
2.  Создайте виртуальное окружение (если еще не создано) и активируйте его.
3.  Установите необходимые зависимости командой:
    ```bash
    pip install numpy pandas matplotlib seaborn scikit-learn xgboost catboost joblib
    ```

### 2. Запуск блокнотов
Все исходные файлы находятся в каталоге `src/`.

* **Задание 1-2:**
    * `lab8-task1_2.ipynb`: Классификация дохода населения. Использование алгоритма случайного леса (RandomForestClassifier) на наборе данных Adult для предсказания уровня дохода. Подбор гиперпараметров с помощью `GridSearchCV`.
* **Задание 3:**
    * `lab8-task3.ipynb`: Сравнение классификаторов. Использование конвейеров (Pipeline) для сравнения различных алгоритмов (KNN, SVM, Decision Tree, Logistic Regression, Random Forest) на наборе данных Iris.
* **Задание 4:**
    * `lab8-task4.ipynb`: Регрессия на данных о диабете. Сравнение и настройка моделей регрессии (SVR, DecisionTreeRegressor, LinearRegression, RandomForestRegressor) на наборе данных Diabetes.
* **Задание 5:**
    * `lab8-task5.ipynb`: Обработка пропусков и бэггинг. Сравнение Logistic Regression, Random Forest и BaggingClassifier на данных кредитного скоринга с предварительным заполнением пропущенных значений.
* **Задание 6:**
    * `lab8-task6.ipynb`: Прогнозирование задержек рейсов. Применение градиентного бустинга (XGBoost и CatBoost) для решения задачи бинарной классификации задержек вылетов на основе набора данных Flight Delays.
